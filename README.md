# Lux Detail — Website

Sitio web profesional para **Lux Detail**, servicio de detailing premium en Chihuahua, México.

## Tech Stack

- **Next.js 16** (App Router) + TypeScript
- **Tailwind CSS v4**
- **Framer Motion** (animaciones)
- **Lucide React** (iconos)
- **Google Fonts** (Cormorant Garamond + DM Sans)

## Setup

```bash
# Instalar dependencias
npm install

# Desarrollo local
npm run dev

# Build de producción
npm run build
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Secciones

- Hero (pantalla completa con imagen de fondo)
- Servicios (6 tarjetas de servicios)
- Nosotros / ¿Por qué Lux Detail?
- Galería (grid de fotos)
- Contacto (formulario + WhatsApp)
- Footer
